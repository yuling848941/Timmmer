音频文件说明 / Audio Files Instructions
=====================================

本目录用于存放计时器超时提示音频文件。
This directory is for storing timer timeout alert audio files.

支持的音频文件格式 / Supported Audio Formats:
- WAV (推荐 / Recommended)
- MP3
- WMA

音频文件命名 / Audio File Naming:
- timeout_alert_chinese.wav - 中文语音提示
- timeout_alert_english.wav - 英文语音提示

使用说明 / Usage Instructions:
1. 将音频文件放置在此目录下
   Place audio files in this directory

2. 确保文件名正确匹配上述命名规则
   Ensure file names match the naming convention above

3. 在程序的音频设置中启用音频提示
   Enable audio alerts in the program's audio settings

4. 选择所需的语言（中文/英文）
   Select the desired language (Chinese/English)

注意事项 / Notes:
- 如果音频文件不存在或播放失败，程序将播放系统默认提示音
  If audio files don't exist or playback fails, the program will play system default beep

- 建议音频文件时长不超过5秒，以免影响用户体验
  Recommended audio duration should not exceed 5 seconds for better user experience

- 音频文件应清晰易懂，音量适中
  Audio files should be clear, understandable, and at moderate volume
