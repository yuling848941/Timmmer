# 更新日志 / Changelog

本文档记录了Timer计时器应用程序的所有重要更改。

## [1.2.1] - 2026-04-24

### 优化 / Optimization
- 🔗 **GitHub 集成**
  - 在“关于”对话框中添加了项目主页链接及直接跳转按钮。
  - 在右键菜单中新增了 GitHub 快捷访问项。

## [1.2.0] - 2026-04-24

### UI 现代化重构 / UI Modernization
- ✨ **全量对话框现代化**
  - 将所有对话框（设置时间、预设编辑、音频、外观、关于）重构为基于 `WS_EX_LAYERED` 的自定义渲染界面。
  - 移除了所有原生 Windows 控件，实现了完全自定义的视觉风格，与主程序完美融合。
- ⌨️ **行内编辑系统**
  - 在“设置时间”和“预设编辑”对话框中引入了行内编辑功能。
  - 支持键盘直接输入、退格删除、方向键导航及光标闪烁反馈。
- 🌑 **交互视觉增强**
  - 为禁用状态的时间单位添加了置灰和禁用交互的逻辑。
  - 优化了对话框弹出位置逻辑，支持多显示器智能定位。
  - 新增 Toast 提示系统，用于输入校验和错误反馈。

### 核心改进 / Core Improvements
- 🧩 **窗口管理重构**
  - 将庞大的 `timer_window.c` 进一步拆分为 `timer_tray.c` (托盘), `timer_buffer.c` (缓冲管理), `timer_window_utils.c` (辅助工具)。
  - 显著提升了代码的可读性和模块化程度。
- 🛠️ **稳健性提升**
  - 修复了对话框关闭时的潜在内存泄漏和 GDI 句柄泄漏问题。
  - 优化了高 DPI 下的渲染精度和圆角平滑度。

## [1.1.0] - 2026-04-21

### 架构重构 / Architectural Refactoring
- 🏗️ **项目目录结构标准化**
  - 将所有文件按照功能划分为 `src/` (源码), `res/` (资源), `docs/` (文档), `build/` (构建), `scripts/` (脚本)。
  - 规范化了资源引用路径，支持在子目录下进行资源编译。
- 🧩 **对话框模块化拆分**
  - 将庞大的 `timer_dialogs.c` (3200+行) 成功拆分为 8 个独立的模块化文件，提升了可维护性和编译效率。
  - 引入了 `timer_dialog_internal.h` 和 `timer_dialog_common.c` 处理公共 UI 逻辑。

### 功能优化 / Improvements
- 🚀 **构建系统升级**
  - 重写了 `build.bat`，支持一键自动化构建并将产物输出到 `build/` 目录。
  - 引入了 `.gitignore` 规范化 Git 仓库管理，排除二进制垃圾文件。
- ⚙️ **配置系统整合**
  - 将之前零散的 6 个配置文件合并为统一的 `timer_config.ini`，简化了用户配置备份。
- 🎨 **UI/UX 增强**
  - 强化了 Fluent 2 视觉风格，包括更精细的阴影和圆角处理。
  - 新增 **鼠标滚轮缩放** 功能，支持实时调整窗口尺寸。
  - 新增 **双击主窗口切换预设** 功能，提升操作便捷性。

### 文档更新 / Documentation
- 📝 更新了 `README.md` 和 `AGENTS.md`，提供了完整且准确的技术架构说明。
- 📂 整理了所有历史设计文档、PRD 和质检报告至 `docs/` 目录。

## 版本说明 / Version Notes

### 版本号规则 / Versioning
本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/) 规范：
- **主版本号**：不兼容的API修改
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正

### 更新类型说明 / Change Types
- 🆕 **新增 (Added)**：新功能
- 🔄 **变更 (Changed)**：现有功能的变更
- 🗑️ **弃用 (Deprecated)**：即将移除的功能
- ❌ **移除 (Removed)**：已移除的功能
- 🐛 **修复 (Fixed)**：问题修复
- 🔒 **安全 (Security)**：安全性相关的修复
