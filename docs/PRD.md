## 1. 产品概述

### 1.1 产品背景

在现代工作和学习环境中，时间管理变得越来越重要。现有的计时器应用要么功能过于复杂，要么界面不够美观，无法满足用户对简洁、高效、美观的桌面计时工具的需求。

### 1.2 产品愿景

打造一款**极简、美观、高效**的 Windows 桌面计时器，让用户专注于时间管理，不被复杂功能干扰。

### 1.3 产品名称

**Timmmer** - 现代化 Windows 桌面计时器

### 1.4 核心价值

| 价值 | 描述 |
|------|------|
| 简洁 | 无复杂设置，开箱即用 |
| 美观 | 现代化 UI，支持自定义外观 |
| 高效 | 低资源占用，后台静默运行 |
| 可靠 | 稳定运行，配置自动保存 |

## 3. 产品定位

### 3.1 竞品分析

| 产品 | 优势 | 劣势 | Timmmer 差异化 |
|------|------|------|----------------|
| 番茄土豆 | 功能丰富 | 过于复杂 | 专注核心计时 |

### 3.2 产品特色

1. **极简设计** - 只保留核心计时功能
2. **透明背景** - 完美融入任何工作界面
3. **智能预设** - 记忆上次使用时间
4. **多语言音频** - 中英文语音提醒
5. **零学习成本** - 直观操作，无需教程

## 4. 功能需求

### 4.1 功能架构

```
Timmmer
├── 核心计时模块
│   ├── 倒计时功能
│   ├── 正计时功能 (超时后)
│   └── 精确计时 (支持毫秒)
├── 时间设置模块
│   ├── 自定义时间
│   ├── 预设时间
│   └── 时间格式配置
├── 界面显示模块
│   ├── 窗口管理
│   ├── 透明度控制
│   └── 颜色主题
├── 音频提醒模块
│   ├── 系统提示音
│   ├── 嵌入音频
│   └── 自定义音频
└── 配置管理模块
    ├── 本地配置
    └── 状态持久化
```

### 4.2 功能列表

#### P0 - 核心功能 (必须实现)

| ID | 功能 | 描述 | 验收标准 |
|----|------|------|----------|
| F001 | 倒计时 | 设定时间后开始倒计时 | 时间准确，误差不超过 1 秒 |
| F002 | 开始/暂停 | 随时开始或暂停计时 | 暂停后时间保持，恢复后继续 |
| F003 | 重置 | 重置为上次设置的时间 | 一键重置，时间准确 |
| F004 | 时间显示 | 显示剩余时间 | 数字清晰，更新及时 |
| F005 | 计时提醒 | 时间到时发出提醒 | 声音正常播放 |
| F006 | 自定义时间 | 手动输入时、分、秒 | 输入合法，即时生效 |

#### P1 - 增强功能 (重要)

| ID | 功能 | 描述 | 验收标准 |
|----|------|------|----------|
| F101 | 预设时间 | 快速选择常用时间 | 支持至少 3 个预设 |
| F102 | 时间格式 | 自定义显示格式 | 可组合时/分/秒/毫秒 |
| F103 | 透明度调节 | 窗口透明度可调 | 滑动条调节，实时预览 |
| F104 | 颜色主题 | 自定义字体和背景色 | 颜色选择器，实时预览 |
| F105 | 透明背景 | 背景完全透明 | 只保留时间文字可见 |
| F106 | 系统托盘 | 最小化到托盘 | 托盘图标正常显示 |
| F107 | 超时正计时 | 超时后继续正计时 | 模式切换正确，颜色变化 |

#### P2 - 扩展功能 (可选)

| ID | 功能 | 描述 | 验收标准 |
|----|------|------|----------|
| F201 | 多语言 | 中英文界面切换 | 切换即时生效 |
| F202 | 多语言音频 | 中英文语音提醒 | 发音清晰可辨 |
| F203 | 自定义音频 | 导入外部音频文件 | 支持 WAV/MP3 格式 |
| F204 | 字体选择 | 选择显示字体 | 支持系统字体 |
| F205 | 窗口记忆 | 记忆窗口位置和大小 | 重启后位置不变 |
| F206 | 恢复出厂 | 重置所有设置 | 配置清空，恢复默认 |

### 4.3 功能详细说明

#### F001 - 倒计时功能

**用户故事**: 作为用户，我希望设定时间后自动倒计时，以便掌握时间进度。

**详细需求**:
- 支持时、分、秒设置
- 支持毫秒级精度显示
- 倒计时到 0 后触发提醒

**边界条件**:
- 最大设置时间：99 小时
- 最小设置时间：1 秒
- 时间精度：±1 秒/小时

#### F107 - 超时正计时功能

**用户故事**: 作为用户，我希望超时后继续正计时，以便知道超时了多久。

**详细需求**:
- 可选启用/禁用
- 超时后自动切换到正计时模式
- 显示颜色变化 (红底白字)
- 正计时时间计入总时间

**边界条件**:
- 启用后才会切换模式
- 禁用时超时后停止计时
- 重置后恢复原始颜色

## 5. 非功能需求

### 5.1 性能需求

| 指标 | 目标值 | 说明 |
|------|--------|------|
| CPU 占用 | < 1% | 正常计时状态下 |
| 内存占用 | < 50MB | 正常运行时 |
| 启动时间 | < 2 秒 | 冷启动到显示界面 |
| 计时精度 | ±1 秒/小时 | 累积误差 |
| 刷新率 | 30-60 FPS | 时间显示更新 |

### 5.2 兼容性需求

| 平台 | 支持版本 | 优先级 |
|------|----------|--------|
| Windows 11 | 所有版本 | P0 |
| Windows 10 | 版本 1903+ | P0 |
| Windows 8.1 | 所有版本 | P2 |
| Windows 7 | SP1+ | P3 |

### 5.3 可靠性需求

| 需求 | 描述 |
|------|------|
| 崩溃率 | < 0.1% (每千次使用) |
| 配置丢失 | 配置自动保存，重启不丢失 |
| 异常恢复 | 程序异常关闭后可正常重启 |

### 5.4 安全性需求

| 需求 | 描述 |
|------|------|
| 文件访问 | 只访问用户目录下的配置文件 |
| 音频文件 | 验证自定义音频文件路径 |

### 5.5 可用性需求

| 需求 | 描述 |
|------|------|

## 6. 技术架构

### 6.1 技术栈

| 层次 | 技术 | 说明 |
|------|------|------|
| 开发语言 | C (C99 标准) | 纯 C 语言，无 C++ |
| 图形界面 | Win32 API | 原生 Windows 窗口 |
| 图形渲染 | GDI | 双缓冲绘制 |
| 音频播放 | winmm.dll | Windows 多媒体 API |
| 配置存储 | INI 文件 | Windows API 读写 |
| 编译工具 | MinGW-w64 | GCC 编译器 |

```

                      │
         ┌────────────┼────────────┐
         │            │            │

         │            │            │
    timer_ui     timer_dialogs  timer_embedded_audio
         │            │
    timer_fonts  timer_config
         │
    timer_embedded_fonts
```

## 8. 数据设计

### 8.1 核心数据结构

```c
typedef struct {
    HWND hMainWnd;              // 主窗口句柄
    int seconds;                // 剩余秒数
    BOOL isRunning;             // 是否正在运行
    BOOL isTimeUp;              // 时间是否到达
    UINT_PTR timerId;           // 计时器 ID
    
    // 时间显示设置
    BOOL showHours;
    BOOL showMinutes;
    BOOL showSeconds;
    BOOL showMilliseconds;
    
    // 外观设置
    COLORREF fontColor;
    COLORREF backgroundColor;
    BYTE windowOpacity;
    BOOL transparentBackground;
    
    // 超时正计时
    BOOL enableOvertimeCount;
    BOOL isInOvertimeMode;
    COLORREF originalFontColor;
    COLORREF originalBackgroundColor;
    
    // 预设时间
    int presetTimes[10];
    int presetCount;
    
    // ... 更多字段见 timer_types.h
} TimerState;
```

### 8.2 状态流转

```
     │              │            │
     │              ▼            │

     │              │
     ▼              ▼
```
