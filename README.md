# 🕐 Timer - 现代化桌面计时器

[English](README_EN.md) | 简体中文


一个功能丰富、界面美观的 Windows 桌面计时器应用程序，支持倒计时、正计时、多语言、自定义外观和音频提示。

## ✨ 主要特性

### 🎯 核心功能
- **精确计时**: 支持小时、分钟、秒、毫秒级精度
- **倒计时模式**: 设定时间后自动倒计时
- **正计时模式**: 超时后继续正向计时
- **预设时间**: 快速选择常用时间设置
- **系统托盘**: 最小化到系统托盘，支持图标实时预览时间和状态
- **窗口缩放**: 支持通过鼠标滚轮实时缩放窗口大小
- **圆角与阴影**: 现代化的 Fluent 2 视觉风格，支持阴影和圆角

### 🌍 多语言支持
- **中文界面**: 完整的中文本地化
- **英文界面**: 国际化英文界面
- **一键切换**: 实时语言切换，无需重启

### 🎨 外观定制
- **字体选择**: 支持系统字体和自定义字体
- **颜色定制**: 自由选择文字和背景颜色
- **透明背景**: 支持完全透明背景模式
- **现代UI**: 圆角窗口、阴影效果
- **实时预览**: 设置时即时预览效果

### 🔊 音频提示
- **多语言音频**: 中文/英文语音提示
- **自定义音频**: 支持导入自定义音频文件
- **智能回退**: 音频文件缺失时自动使用备用方案
- **音频测试**: 设置时可预览音频效果

## 🚀 快速开始

### 系统要求
- **操作系统**: Windows 10/11 (推荐) 或 Windows 7/8
- **内存**: 最少 50MB 可用内存
- **存储**: 约 10MB 磁盘空间

### 安装方法

#### 方法一：直接运行 (推荐)
1. 下载 `timer_audio_optimized.exe`
2. 双击运行即可使用
3. 首次运行会自动创建配置文件

#### 方法二：从源码编译
```bash

git clone https://github.com/your-repo/timer.git
cd timer

# 使用项目自带的脚本编译
build.bat

# 运行程序
./build/timer_audio_optimized.exe
```

## 📖 使用指南

### 基本操作

#### 1. 设置时间
- **右键菜单**: 右键点击计时器 → "设置时间"
- **快捷操作**: 双击计时器窗口
- **预设时间**: 右键菜单 → "预设时间" → 选择常用时间

#### 2. 开始计时
- **开始/暂停**: 右键菜单 → "开始/暂停"
- **重置**: 右键菜单 → "重置"

#### 3. 快捷交互
- **快速切换预设**: **双击** 计时器主窗口可按顺序循环切换预设时间
- **缩放窗口**: 在主窗口滚动 **鼠标滚轮** 可按比例放大或缩小窗口
- **移动窗口**: 按住主窗口任意位置即可拖动

#### 3. 自定义外观
1. 右键菜单 → "外观设置"
2. 选择字体、颜色、透明度
3. 实时预览效果
4. 点击"确定"保存设置

#### 4. 音频设置
1. 右键菜单 → "音频设置"
2. 启用音频提示
3. 选择语言或自定义音频文件
4. 点击"测试音频"预览效果

### 高级功能

#### 预设时间管理
- **添加预设**: 在预设时间对话框中输入分钟数，点击"添加"
- **删除预设**: 选择预设时间，点击"删除"
- **修改预设**: 选择预设时间，输入新值，点击"修改"

#### 时间显示格式
- **自定义显示**: 可选择显示小时、分钟、秒、毫秒
- **灵活组合**: 根据需要组合不同的时间单位

#### 透明背景模式
- **完全透明**: 启用透明背景，只显示时间文字
- **抗锯齿优化**: 透明模式下自动禁用字体抗锯齿，确保清晰显示

## 🔧 配置文件

程序会在用户目录下创建一个统一的配置文件：

```
%USERPROFILE%/
└── timer_config.ini     # 所有功能、外观、预设和窗口状态的统一配置文件
```

#### 综合配置 (timer_config.ini) 结构示例

```ini
[Format]
ShowHours=1                    # 显示小时 (0/1)
ShowMinutes=1                  # 显示分钟 (0/1)
ShowSeconds=1                  # 显示秒 (0/1)
ShowMilliseconds=0             # 显示毫秒 (0/1)

[General]
Language=0                     # 界面语言 (0:中文, 1:英文)

[Appearance]
FontColor=16777215             # 字体颜色 (COLORREF)
BackgroundColor=0               # 背景颜色 (COLORREF)
WindowOpacity=255              # 窗口透明度 (0-255)
FontName=Arial                 # 选中的字体名称
TransparentBackground=0        # 背景完全透明模式 (0/1)

[Audio]
EnableAudioAlert=1             # 开启音频提示 (0/1)
UseChineseAudio=1              # 使用中文语音 (0/1)
UseCustomAudio=0               # 使用自定义音频 (0/1)
CustomAudioPath=               # 自定义音频文件路径

[Presets]
Count=3                        # 预设时间数量
Preset0=10                     # 预设1 (分钟)
Preset1=15                     # 预设2 (分钟)
Preset2=20                     # 预设3 (分钟)

[Timer]
EnableOvertimeCount=0          # 开启超时正计时 (0/1)
LastUsedTime=600               # 上次设置的时间 (秒)

[Window]
X=100                          # 窗口位置 X
Y=100                          # 窗口位置 Y
Width=300                      # 窗口宽度
Height=150                     # 窗口高度
```

## 🔍 故障排除

### 常见问题

#### Q: 程序无法启动
**A**: 检查系统是否支持，确保有足够的内存空间

#### Q: 音频无法播放
**A**: 
1. 检查音频设置是否启用
2. 确认系统音频设备正常
3. 尝试不同的音频语言选项

#### Q: 字体显示异常
**A**:
1. 尝试重置字体设置
2. 检查系统字体是否完整
3. 使用默认字体设置

#### Q: 配置丢失
**A**:
1. 检查用户目录权限
2. 手动删除配置文件重新生成
3. 使用"恢复出厂设置"功能

### 性能优化建议

1. **减少重绘**: 不显示毫秒时性能更好
2. **字体缓存**: 避免频繁更换字体
3. **透明模式**: 在低性能设备上可能影响性能
